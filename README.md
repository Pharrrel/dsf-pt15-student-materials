# DSF-PT15 Student Materials

Class notebooks and datasets for the **Data Science Foundations — Part-Time 15** cohort at Moringa School.

## Getting Started

### 1. Clone this repo (first time only)

```bash
git clone https://github.com/ds-moringa-datanerds/dsf-pt15-student-materials.git
cd dsf-pt15-student-materials
```

### 2. Pull the latest materials (before each class)

```bash
git pull
```

This downloads any new notebooks or datasets added since your last pull.

## Repo Structure

```
dsf-pt15-student-materials/
├── data/                   # Datasets used across notebooks
│   └── financial_inclusion_africa.csv
├── Week3/                  # Week 3 notebooks
│   └── W3D2_TypesOfData_Student.ipynb
└── README.md
```

New folders will be added each week — just `git pull` to stay up to date.

## Opening a Notebook

1. Open **Anaconda Navigator** and launch **Jupyter Notebook**
2. Navigate to the folder where you cloned this repo
3. Open the `.ipynb` file for your session

## Course Website

[ds-moringa-datanerds.github.io](https://ds-moringa-datanerds.github.io) — schedule, slides, and recordings.

## Questions?

Post in the **DSF-PT15** Mattermost channel or ask during office hours.
